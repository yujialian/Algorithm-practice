### Template literals:

### Variables:
  const: Constant value
  let: Exist in the inner most curly braces surrounding it.
  var: Exist all the way in the function if it's defined in the function

### Arrow functions: Defined without the word of function
  () => {
    alert('hello world');
  }
  x => {
    alert(x);
  }
