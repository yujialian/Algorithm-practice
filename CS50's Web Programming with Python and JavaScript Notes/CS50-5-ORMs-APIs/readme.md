### APIs
```
Application programming interfaces.
Think of as an protocol for communication between difference application, different part of same application.
(Share information from each other.)

JSON: JavaScript object notation: Simple way of representing information in a way that human readable and computer readable.

Why we want API: Machine readable.

HTTP methods:(Different ways to make request to web server.)
GET: Retrieve resource
POST: Create a new resource
PUT: Replace a resource
PATCH: Update a resource
DELETE: Delete a resource

Status codes:
200 ok
201 created
400 bad requests
403 Forbidden
404 Not found
405 Method not allowed
422 Unprocessable Entity
...

Python requests library:
Make http requests in python.

```
